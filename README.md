### ⚽ Football Player Valuation Engine
> *Scikit-learn · XGBoost · Streamlit · Feature Engineering*

An end-to-end ML regression system that estimates football player market value with an **R² score of ~0.62**.
- Processed **30,000+ player records** and **100,000+ data points** across multiple files
- Engineered high-impact features: `Goals per90`, `league rank`, `log transform`
- Deployed as a live **Streamlit web app** for real-time valuation predictions
